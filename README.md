###Installation

1. Open the terminal, in a directory of your choice, and clone the project:
```bash
  git clone git@github.com:flavio-bianchetti/go-todo-api.git
```

2. Enter the project directory:
```bash
  cd go-todo-api
```
3. Install dependencies:
```bash
  npm install
```
